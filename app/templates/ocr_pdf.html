O que é OCR?

OCR (Reconhecimento Óptico de Caracteres) extrai texto de PDFs escaneados ou imagens. Ideal para documentos digitalizados que não possuem texto selecionável.

Configurar Tesseract OCR
Método Recomendado: Copie os arquivos do Tesseract diretamente para a pasta do projeto.
Baixar o Tesseract
  1. Acesse: https://github.com/UB-Mannheim/tesseract/wiki
  2. Baixe o instalador mais recente (ex: tesseract-ocr-w64-setup-5.x.x.exe)
  3. Execute o instalador
  4. Importante: Marque os idiomas:
    • ☑ Portuguese
    • ☑ English
Copiar para o projeto
  1. Após instalar, abra o Explorador de Arquivos
  2. Navegue até: C:\Program Files\Tesseract-OCR
  3. Copie todo o conteúdo da pasta (Ctrl+A, Ctrl+C)
  4. Cole na pasta tesseract do projeto (Ctrl+V)
Estrutura esperada:
VestasLovePDF/
└── tesseract/
    ├── tesseract.exe
    ├── tessdata/
    │   ├── eng.traineddata
    │   ├── por.traineddata
    │   └── ...
    └── *.dll (arquivos de biblioteca)
Verificar

Após copiar os arquivos, feche e reabra esta página. O sistema detectará o Tesseract automaticamente.

Alternativa: Linux/macOS
Linux (Ubuntu/Debian)
sudo apt-get install tesseract-ocr tesseract-ocr-por tesseract-ocr-eng
macOS
brew install tesseract tesseract-lang
